Phase 3 — DSA Patterns

THE 75 + enterprise classics · ~120 problems · 14 pattern tutorials · JavaScript solutions · calibrated for enterprise-tier loops (Optum, JPMC, Goldman & similar)

Pattern fluency over problem count. Enterprise-tier loops — banks, fintech, healthcare and large product companies (Optum, JPMC, Goldman Sachs and their peers) — draw overwhelmingly from easy–medium classics — what's graded at 7 YOE is speed, clean code, and narrating your thinking. Each tutorial teaches one pattern: how to recognize it from the problem statement, the reusable code template, then curated problems in rising difficulty. Every problem has a full crafted statement with examples (attempt it right on the page), a link to run it on LeetCode, a hint reveal, and a commented JS solution with complexity and follow-ups. Attempt every problem for 25–30 minutes before opening the solution — reading solutions feels like progress but isn't.

EASY MEDIUM HARD — badges on every problem · ★ = THE 75 (the core must-do list) · hards are labeled Stretch (rarely asked at enterprise-tier companies — do them last, not first)
