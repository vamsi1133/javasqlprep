# Full-Stack Interview Prep

Senior-level (7 YOE) interview preparation courses — in-depth HTML tutorials with hundreds of interview Q&As, hands-on exercises, and a capstone project.

**📖 Read the courses:** https://vamsi1133.github.io/javasqlprep/

## Phase 1 — Java · Spring Boot · PostgreSQL (16 tutorials)

- **Core Java:** OOP & collections internals, modern Java 8→21, concurrency, JVM
- **Spring Boot:** DI/core, REST APIs, JPA/Hibernate, Security/JWT, AOP & testing, microservices
- **PostgreSQL:** SQL fluency, schema design, indexing & EXPLAIN, transactions/MVCC, ops & JSONB
- **Capstone:** Order-Management API (Spring Boot 3.x + Postgres 16, 12 milestones)

## Phase 2 — Frontend Engineering (18 tutorials)

- **HTML/CSS:** accessibility & ARIA, CSS internals (specificity, stacking contexts, Flexbox/Grid), modern CSS (:has(), container queries, layers) & styling architecture
- **JS/TS:** event loop & closures internals, implementation drills (debounce, Promise.all, emitter…), advanced TypeScript
- **React:** rendering internals & Fiber, hooks deep dive, performance, React 18/19 (RSC, Actions, compiler), Redux Toolkit & state management, component patterns, testing (RTL/MSW/Playwright), Next.js
- **At scale:** micro-frontends, build tooling & modules, Core Web Vitals, security/networking, UI/UX engineering

## Phase 3 — DSA Patterns (14 tutorials, ~120 problems)

Pattern fluency over problem count — THE 75 plus the enterprise-tier classics, all solutions in JavaScript.

- **Core patterns:** arrays & hashing, two pointers, sliding window, stacks & monotonic stacks, binary search
- **Data structures:** linked lists (incl. LRU Cache), trees & BST, heaps & top-K (incl. a MinHeap you write once)
- **Recursion & graphs:** backtracking, graphs (BFS/DFS, topological sort, union-find)
- **DP & the rest:** 1D DP, greedy & intervals, 2D DP, tries/bits/matrix math

Every problem: a full statement with examples you can attempt on the page, a LeetCode link, a derivation of the
solution from the brute force, and a click-to-reveal commented JS solution with a trace, complexity, and follow-ups.

## Phase 4 — System Design (27 tutorials)

Covers both rounds you can actually get: FAANG-grade distributed design, and the pragmatic
enterprise/fintech flavour (payments, idempotency, inventory contention).

- **Foundations:** the interview method · networking & delivery · data & storage engines
  (incl. B-tree vs LSM) · partitioning & **consistent hashing** · coordination, consensus &
  distributed transactions (2PC vs saga, outbox) · caching, queues, resilience & observability
- **Backend case studies** — one per *archetype*, not per company: URL shortener ·
  distributed cache/KV store · rate limiter · news feed · chat · notifications & scheduler ·
  payments & orders · ticket booking · geospatial/nearby · video streaming · web crawler & search ·
  streaming aggregation
- **Frontend system design:** the RADIO method · news feed UI · autocomplete/typeahead ·
  component library/design system · real-time collaborative UI
- **Low-level design (Java):** SOLID & the patterns that show up · case studies (parking lot,
  elevator, BookMyShow)
- **Drills:** back-of-envelope estimation · trade-off flashcards

Each case study is a worked transcript of a strong 45-minute answer: requirements → estimation with
real numbers → API → schema → architecture → the deep dive the interviewer pushes into → trade-offs
→ follow-ups.

## Phase 5 — Behavioral & Mock Interviews (8 tutorials)

The round that most often decides the offer once your coding is good enough. Calibrated for both
FAANG-grade values loops (Amazon Leadership Principles, Meta, Google) and the pragmatic
enterprise/fintech behavioral bar (Optum, JPMC, Goldman).

- **Method & material:** what's actually graded + the STAR/SAR method · building a reusable
  8–10 story bank that covers the whole round
- **Question archetypes:** leadership/ownership/influence · conflict/feedback/collaboration ·
  failure/mistakes/ambiguity · "tell me about yourself", why-company & questions to ask
- **Company calibration:** all 16 Amazon Leadership Principles mapped to your stories · the Bar
  Raiser · Meta/Google at awareness level · the enterprise/fintech bar
- **Execution:** delivery mechanics · a 40-question mock bank · salary negotiation

Each tutorial: concepts in depth with runnable code (where apt), T1/T2/T3 interview-priority badges, pitfalls & senior-level nuances, click-to-reveal interview Q&A for self-testing, and exercises with solutions.
