# Java · Spring Boot · PostgreSQL — Interview Prep

Senior-level (7 YOE) full-stack interview preparation course — 16 in-depth HTML tutorials with ~400 interview Q&As, hands-on exercises, and a capstone project.

**📖 Read the course:** https://vamsi1133.github.io/javasqlprep/

## Contents

- **Block A — Core Java:** OOP & collections internals, modern Java 8→21, concurrency, JVM
- **Block B — Spring Boot:** DI/core, REST APIs, JPA/Hibernate, Security/JWT, AOP & testing, microservices
- **Block C — PostgreSQL:** SQL fluency, schema design, indexing & EXPLAIN, transactions/MVCC, ops & JSONB
- **Block D — Capstone:** Order-Management API (Spring Boot 3.x + Postgres 16, 12 milestones)

Each tutorial: concepts in depth with runnable code, T1/T2/T3 interview-priority badges, pitfalls & senior-level nuances, click-to-reveal interview Q&A for self-testing, and exercises with solutions.

## Local use

No build needed — open `phase1-java-spring-postgres/index.html` in a browser.
