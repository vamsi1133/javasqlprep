Phase 1 — Java · Spring Boot · PostgreSQL

Senior-level (7 YOE) full-stack interview prep · frontend-expert pivoting to full-stack · FAANG + product startups

Work through the tutorials in order. Each one is a self-contained deep-dive: concepts explained like a tutorial, runnable code, pitfalls, senior-level nuances, and the most-asked interview questions with click-to-reveal model answers so you can self-test.

T1 — must be airtight (asked in almost every senior interview) T2 — senior differentiator T3 — awareness is enough

Block A — Core Java (~5–6 days)

  • 01Java OOP & Collections Internals equals/hashCode contract · String pool · interfaces vs abstract · generics & PECS · HashMap internals · ConcurrentHashMap · fail-fast iterators
  • 02Modern Java (8 → 21) Lambdas · Stream API deep-dive · Optional · records · sealed classes · pattern matching · virtual threads (Loom)
  • 03Concurrency & Multithreading Thread lifecycle · synchronized vs volatile · thread pools · CompletableFuture · deadlocks · Java Memory Model · CAS & atomics
  • 04JVM Internals Heap/stack/metaspace · GC (G1, ZGC) · memory leaks · class loading · JIT · common JVM flags & OOM types

Block B — Spring Boot (~6–7 days)

  • 05Spring Core, DI & Boot Fundamentals IoC/DI · bean lifecycle & scopes · circular deps · auto-configuration mechanics · starters · config & profiles · Actuator
  • 06REST API Layer Controllers · DTO pattern · validation · @ControllerAdvice & ProblemDetail · REST design · filters vs interceptors · CORS
  • 07Spring Data JPA & Hibernate Entity mapping · relationships · N+1 problem & fixes · @Transactional deep-dive · entity states · locking · HikariCP
  • 08Spring Security Filter chain · JWT end-to-end · access/refresh tokens · OAuth2 + PKCE · method security · CSRF for stateless APIs
  • 09AOP, Proxies & Testing JDK vs CGLIB proxies · aspects & pointcuts · JUnit 5 · Mockito · @WebMvcTest / @DataJpaTest · Testcontainers
  • 10Microservices & Ecosystem Monolith vs microservices · resilience patterns · Kafka fundamentals · Saga & outbox · API gateway · observability

Block C — PostgreSQL (~4–5 days)

  • 11SQL Fluency Joins · GROUP BY/HAVING · NULL traps · window functions · CTEs · UPSERT · the classic live-coding SQL questions
  • 12Schema Design Normalization 1NF→3NF · keys & constraints · UUID vs BIGSERIAL · data types done right · live schema-design exercises
  • 13Indexing & Query Performance B-tree mechanics · composite & covering indexes · EXPLAIN ANALYZE · why the planner ignores your index · slow-query diagnosis
  • 14Transactions, Isolation & MVCC ACID · isolation levels & anomalies · MVCC internals · FOR UPDATE / SKIP LOCKED · deadlock avoidance · idempotency
  • 15Operations & Advanced Postgres VACUUM & bloat · connection pooling · replication · partitioning · JSONB vs MongoDB question · full-text search · materialized views

Block D — Capstone (~2–3 days)

  • 16Capstone: Order-Management API Build a Spring Boot + Postgres service exercising every T1 topic — entities, validation, @Transactional, JWT, N+1 hunt, Testcontainers
